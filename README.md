## 📚진행 방식
- 각 주차 담당이 문제를 선정한다.
- 각자 문제를 풀고 매주 월요일 17시까지 PR로 코드 공유한다.
- 스터디 전까지 코드 리뷰를 진행한다.(PR에 댓글 남기는 형식)
- 월요일 17시까지 모든 문제의 풀이를 올리지 못한 경우 벌금 부과
  - 한 문제 1000원, 두 문제 3000원, 세 문제 이상 5000원
  - 공지에 장부 기록 후 대면으로 만나서 차감.
<br/>

## 😸 Github Rule
- 본인이름/폴더구조 생성 후, 풀이 코드 제출 (이름 폴더 이하 구조는 각자 취향에 맞게)<br/>
    ex) 이주은/Greedy/prob1.java
- 커밋 메시지 말머리는 ps, refactor, docs로 통일. 그 밖의 경우에는 자유롭게.
    ex) ps: 백준 1931번 회의실 배정
